# Theo Bedankt - Music Player

A beautiful music player web application.

## Deployment to Vercel

### Option 1: Deploy via Vercel Dashboard (Recommended)

1. Go to [vercel.com](https://vercel.com) and log in
2. Click "Add New Project"
3. Import your Git repository OR drag and drop the project folder
4. Vercel will automatically detect the static site and deploy

### Option 2: Deploy via GitHub

1. Push this project to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and log in
3. Click "Add New Project"
4. Import your GitHub repository
5. Vercel will automatically deploy on every push

### Option 3: Update Node.js and use CLI

If you want to use the CLI:
1. Update Node.js to version 18 or higher
2. Run: `vercel --yes`

## Project Structure

- `index.html` - Main application file
- `songs/` - Audio files directory
- `vercel.json` - Vercel configuration

## Features

- Starts with song 4
- Next/Previous navigation buttons
- Beautiful, modern UI
- Responsive design
